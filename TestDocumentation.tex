\section{Introduction}
This document outlines the test cases for the mail deletion functionality of the mail provider (Gmail). The test cases are designed to ensure that mail deletion works as expected across various scenarios.

\section{Test Case Format}
Based on the guidelines from Section 1 of GUIDE.MD, each test case includes:
\begin{itemize}
\item \textbf{Test Case ID}: Unique identifier
\item \textbf{Objective}: Clear statement of what the test aims to achieve
\item \textbf{Preconditions}: Initial state required before execution
\item \textbf{Test Inputs}: Specific data fed into the system
\item \textbf{Execution Steps}: Sequence of events to carry out the test
\item \textbf{Expected Results}: Anticipated output/state transition
\item \textbf{Actual Results}: Observed behavior after execution
\end{itemize}
